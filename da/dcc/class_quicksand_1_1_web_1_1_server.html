Iterate over directories pointed by the path to add all files as resources

File System:
myTestDir
|- dir1
| |- fileA.txt
| – fileB.txt
|- fileC.txt
– fileD.txt
server.AddResource("/test", "myTestDir");
Will result in the add of resources:
/test/dir1/testa.txt
/test/dir1/testb.txt
/test/testc.txt
/test/testd.txt

Parameters
urlPath of the file in the server. Should start with a /
pathPath of the file to add
preLoadSpecify if we should load in memory the file content. False by default
